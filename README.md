🔍 Project Overview

This project focuses on Exploratory Data Analysis (EDA) of television products listed on Flipkart to understand market trends, pricing patterns, and customer preferences.

The analysis is based on real-world data collected through web scraping, followed by data cleaning, visualization, and insight generation to support business decision-making.

🎯 Problem Statement

Understanding consumer behavior in the online television market by analyzing:

Best-performing brands
Price distribution across segments
Impact of features like screen size, resolution, and smart functionality

The goal is to derive insights that help improve product positioning, pricing strategy, and customer satisfaction.

🛠️ Tools & Technologies Used:

Python
Pandas, NumPy → Data Cleaning & Processing
Matplotlib, Seaborn → Data Visualization
BeautifulSoup, Requests, Regex → Web Scraping
Jupyter Notebook → Analysis

Data scraped from Flipkart using Python

🧹 Data Cleaning & Preparation

Removed null and inconsistent values
Standardized numerical fields (Price, Ratings, Screen Size)
Cleaned text data using Regular Expressions
Converted unstructured product descriptions into usable features
📊 Exploratory Data Analysis
🔹 Univariate Analysis
Premium brands like Samsung, Philips, TCL have higher prices
Budget brands like Foxsky, Thomson dominate lower price range
Majority of TVs fall within ₹20,000 – ₹40,000 range
Screen size and price show a clear upward trend
🔹 Bivariate Analysis
Strong relationship between screen size and price
Brands like LG, Xiaomi, Samsung show higher ratings
Ultra HD (4K) TVs dominate across most brands
Mid-range TVs receive better ratings compared to premium extremes
📈 Key Insights
📊 Most popular TVs are priced between ₹20,000 – ₹60,000
📺 32–43 inch TVs are most preferred by customers
🌟 Ultra HD TVs dominate the market demand
💰 Mid-range TVs provide the best value for money
📉 Higher price does not always guarantee higher customer satisfaction
⭐ TVs with better ratings tend to receive more reviews (high engagement)
💼 Business Recommendations
Focus on mid-range segment to maximize sales volume
Promote Ultra HD Smart TVs as they drive demand
Optimize pricing strategies based on screen size and features
Highlight high-rated products to increase customer trust and conversions
